MySQL 8.0 버전에서 root 비밀번호 변경하기

MySQL root 비밀번호를 알고 있는 상태에서 변경하려는 경우 (초기화 아님)

유태건

Nov 26, 2019·2 min read

OS : Windows 10(x64)
MySQL : 8.0.18 (MySQL Community Server)

  1. 비밀번호를 변경합니다.
ALTER user 'root'@'localhost' IDENTIFIED WITH mysql_native_password BY '변경할 비밀번호';

기존(5.7 이상 버전)에는 UPDATE mysql.user SET authentication_string=password(‘변경할 비밀번호’) WHERE user=’root’ 방식을 썼는데, 지금 실행하면 에러가 발생합니다.

2. 변경 사항을 저장합니다.

FLUSH PRIVILEGES;

3. MySQL을 종료한 후 다시 로그인 해 봅니다.

exitmysql -u root -p
(비밀번호 입력)

성공적으로 비밀번호가 변경된 것을 확인할 수 있습니다.

참고 : https://to-dy.tistory.com/58 (todyDev)
